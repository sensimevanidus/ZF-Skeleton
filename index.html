ZF-Skeleton by sensimevanidus

Zend Framework skeleton with Doctrine 1.2 support. It contains different module skeletons (e.g. normal website, Facebook application, REST API) and lets you build effective applications in a short time.

Download

You can download this project in either zip or tar formats.
